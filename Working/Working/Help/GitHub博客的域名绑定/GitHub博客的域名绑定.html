GitHub博客的域名绑定

@2015-02-25

暂时好像只能绑定一个。

比较容易成功的是blog.hypergroups.cn

如果要绑定www.hypergroups.cn或hypergroups.cn

在CNAME里填写www.hypergroups.cn

其中a记录的IP处网上会搜索到三种：
一种是192的是目前使用的，
一种是207的，可能是以前的，
还有一种是说ping hypergroups.github.io获得的，比如我是113的。

本来是比较简单的事情，但是我还是花费了些时间尝试，不知道是否是凌晨的时候生效太慢导致。
后经咨询网友，确认了一下，说是他能打开，我自己电脑打不开，后来我也能打开了。
因此，必要时可能要清理DNS缓存

ipconfig /flushdns
